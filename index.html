3D Interactive coin with animated particle system background

Description:
Presidential dollar coin of George Washington was modeled using Blender and animated using three.js. It also doubles as a display piece as the coin will slowly rotate when idle. I tried to strike a balance between resolution and performance so it can still be enjoyed on low performance systems. Interactive on on computers, smartphones, and tablets. I would like to put an original design on it if accepted as a creator.

Future creator plans:
I plan to make a few more particle system backgrounds or add effects to the coin.

How to Interact:
Click/Tap the coin to spin for heads or tails.
Click/Tap and drag to view the coin from different angles.
Scroll to zoom.

Interactive rollable dice series using physics engine
Animated dice rendered using threejs to generate an n sided die and ammo.js physics engine to simulate the roll. The user would be able to configure their dice in real time to generate however many sides they want. look into if three js supports gyroscopic motion detection on a phone. If supported users can shake their phones to roll dice.
